Projects

Quantative Stock Selection

I did research with the Ying Wu College of Computing here at NJIT. Specifically, I did a project that looked into quantitatively making investment decisions. This project included a variety of competencies. I created a backtest engine that would allow me to test strategies that were both long and short on equities. I also created a variety of strategies that analyzed market data to make trading decisions. Some of my strategies beat the market.

Data Analytics / Data Engineering

In various roles I have worked the entire stack of data analytics from meeting clients to gather requirements and see what problems can be solved with data, to setting up data mining and staging areas, to creating dashboards to allow myself and others to analyze data. Data is in abundant quantities all around us, and I have the skills necessary to separate white noise from directly actionable (and therefore profitable) insight.

Makerspace Access Control System

I worked on a team that created an access control system for the NJIT Makerspace. This system included client devices that users interacted with and a server that handled all of the logic. I was responsible for developing an administrator portal for employees to manage users, creating and maintaining the database, and creating an API through which the client devices could get the information they needed to allow user interaction.
